Элемент управления графиком:

Диапазон х:

"от:" - поле для ввода начальной координаты графика.

"до:" - поле для ввода конечной координаты графика.

"Начать построение" - кнопка для начала построения графика.

"Остановить построение" - кнопка для остановки построения графика, очищения таблицы и полей ввода данных.

"Масштаб +" - кнопка увеличения масштаба графика.

"Маштаб -" - кнопка уменьшения масштаба графика.
